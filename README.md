# Shut the Box Game

## Description
Shut the Box is a traditional dice game, often played in pubs, where players aim to 'shut' as many numbered tiles as possible by rolling dice and adding the values together. The goal is to shut all the tiles or achieve the lowest score possible.

## How to Play
1. Players take turns rolling two dice.
2. Based on the total of the dice, players can 'shut' one or more tiles that match the sum.
3. If a player cannot make a valid move, their turn ends.
4. The game continues until all tiles are shut, or no further moves are possible.

## Features
- Roll Dice
- Shut Tiles
- Count Score
